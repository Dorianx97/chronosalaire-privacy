🔒 Politique de confidentialité

Heuro — Dernière mise à jour : janvier 2025

Heuro respecte votre vie privée. Cette politique explique de manière transparente comment vos données sont traitées dans le cadre de l'utilisation de l'application.

1. Données collectées par l'application

Heuro ne collecte aucune donnée personnelle. L'intégralité de vos données (heures travaillées, paramètres de salaire, préférences) est stockée exclusivement sur votre appareil et n'est jamais transmise à nos serveurs ni à des tiers.

2. Publicités (Google AdMob)

Heuro utilise Google AdMob pour afficher des publicités. AdMob est susceptible de collecter les données suivantes :

  • Identifiant publicitaire (IDFA sur iOS, GAID sur Android)
  • Données d'interaction avec les publicités
  • Informations sur l'appareil (modèle, version OS)

Ces données sont utilisées pour afficher des publicités pertinentes et mesurer leur efficacité. Vous pouvez limiter la publicité personnalisée dans les paramètres de votre appareil :

  • iOS : Réglages → Confidentialité → Publicité Apple → Limiter le suivi publicitaire
  • Android : Paramètres → Google → Annonces → Désactiver la personnalisation des annonces

Pour en savoir plus sur la politique de confidentialité de Google : policies.google.com/privacy

3. Achats intégrés (Abonnement "Sans Pub")

L'abonnement "Sans Pub" est géré directement par Apple (App Store) ou Google (Play Store). Heuro ne traite ni ne stocke aucune information de paiement. Ces transactions sont soumises aux conditions générales de vente d'Apple et de Google.

Vous pouvez gérer ou annuler votre abonnement à tout moment depuis :

  • iOS : Réglages → [votre nom] → Abonnements
  • Android : Google Play → Menu → Abonnements

4. Exportation de documents

Les PDF générés et les emails envoyés via la fonction d'export contiennent uniquement les données que vous avez saisies dans l'application. Ces données ne transitent pas par nos serveurs — elles sont partagées directement depuis votre appareil via les services de partage natifs de votre système.

5. Stockage local

Vos données sont stockées localement sur votre appareil via AsyncStorage (stockage local sécurisé). Heuro n'a pas accès à ces données à distance. La désinstallation de l'application entraîne la suppression définitive et irréversible de toutes vos données.

6. Vos droits (RGPD)

Conformément au Règlement Général sur la Protection des Données (RGPD), vous disposez des droits suivants :

  • Droit d'accès : vos données sont visibles directement dans l'application
  • Droit de suppression : désinstallez l'application pour supprimer toutes vos données
  • Droit d'opposition : limitez la publicité personnalisée dans les paramètres de votre appareil
  • Droit de portabilité : exportez vos données via la fonction d'export PDF

7. Modifications de cette politique

Cette politique de confidentialité peut être mise à jour pour refléter des changements dans l'application ou la législation applicable. Les modifications importantes seront notifiées via une mise à jour de l'application sur les stores. La date de dernière mise à jour est indiquée en haut de ce document.
